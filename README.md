# 數學寶可夢對戰 PWA

## 檔案說明
- `index.html` — 主程式（含一般模式＋計時模式）
- `manifest.json` — PWA 設定
- `sw.js` — Service Worker（離線支援）
- `icon-192.png` / `icon-512.png` — 應用程式圖標

## 部署方式（GitHub Pages）

1. 在 GitHub 建立新 repository
2. 上傳所有檔案到 repository 根目錄
3. 到 Settings → Pages → Source 選擇 main branch
4. 等待部署完成，取得網址如 `https://username.github.io/repo-name/`

## 安裝到手機

1. 用手機瀏覽器打開部署後的網址
2. iOS: Safari → 分享 → 加入主畫面
3. Android: Chrome → 選單 → 安裝應用程式

## 功能
- ⚔️ 一般模式：不限時間答題
- ⏱️ 計時模式：5秒限時答題
- 🔥⚡🌊💥 四種招式搭配不同數學題型
- 📈 招式升級系統
- 🌿🔥👻🛡️💀 屬性相剋
- 🗺️ 五種專屬戰鬥場景
- 🐣→🐥→🦅 三階進化系統
- 📱 支援離線遊玩
